# GodGrid
GodGrid to 2D sandbox/symulacja swiata w Pythonie (pygame), w ktorej gracz dziala jak "bog":
edytuje teren, generuje nowe mapy i obserwuje zycie zwierzat.

## Wymagania
- Python 3.11+
- pakiety z `requirements.txt`

## Instalacja
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Uruchomienie
```bash
python src/main.py
```
